GT SUPER KONTROLER 2026 V001F STABLE - KISA KURULUM
====================================================

1) ZIP'i klasore cikart.
2) Icindeki tum dosya ve klasorleri GitHub deposunun ana dizinine yukle.
   Gizli .github klasoru mutlaka yuklenmelidir.
3) GitHub > Actions > Build GT SUPER KONTROLER V001F STABLE > Run workflow.
4) Islem bitince artifact ZIP'ini indir.

UF2 YUKLEME
-----------
P1_SILAH.uf2 -> Oyuncu 1 silah Pico
P2_SILAH.uf2 -> Oyuncu 2 silah Pico
CONTROLER.uf2 -> Kontrol karti Pico

SABIT GP / KLAVYE
-----------------
GP2=1, GP3=2, GP4=3, GP5=4, GP6=5, GP7=6, GP8=7,
GP9=8, GP10=9, GP11=0.

Kontrol kartinda:
GP27=Role 1, GP26=Role 2.

Silah Pico'larda:
GP26/ADC0 ve GP27/ADC1 potansiyometredir.
GP19 fiziksel ac/kapat anahtaridir.

ILK CALISTIRMA
--------------
1) Windows programini ac.
2) Cihazlari Tara / Yenile'ye bas.
3) P1 ve P2 icin 4 kose kalibrasyon yap.
   Program X/Y kanalini ve ters yonleri otomatik belirleyip Pico'ya kaydeder.
4) GP19 ters calisiyorsa anahtari fiziksel KAPALI konuma getirip
   "GP19 KAPALI Konumu Ogret" dugmesine bir kez bas.
5) F5, P1/P2 anahtar konumunu yazilimsal olarak tersler.

Detaylar icin V001F_DEGISIKLIKLER.txt dosyasini oku.
